Segue uma lista dos meus links e programas favoritos.

Alguns sites para jogar:
  • Lichess (Gratuito e com muitos recursos para analisar.)
  • ICC (O que possui mais mestres online do mundo todo.)
  • Chess.com (Para jogar OnLine e para jogar por correspondência.)
  • FICS (Gratuito e muito bom também.)
Ferramentas para jogar e ou analisar:
  • Chessbase (O melhor para Windows, é pago, mas vale a pena)
  • Chessbase Online (O melhor software para se ter no smartphone, pago)
  • SCID (Um "chessbase" pra quem usa Linux)
  • XBoard (Linux. dá pra jogar no FICS)
  • StockFish
Bases de Xadrez:
  • MegaDataBase (pago. O maior Banco de Dados de partidas.)
  • PGN Mentor (para obter modelo de partidas em determinadas variantes) 
  • Chess-db (excelente para buscar partidas do adversário) 
  • Chess-Games (bom pra pesquisar uma ou outra partida clássica rapidamente)
  • Chess-Results (base de torneios)
  • ratings.fide.com (Busca de jogadores) 
  • Follow Chess (Aplicativo Android para ver torneios e partidas em tempo real)
Comunidades:
  • Chess.com
